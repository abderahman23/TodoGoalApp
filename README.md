Description

This is a simple To-Do List (Goal) application developed using React Native and Expo. Users can add their goals or tasks, view them in a list, and delete tasks once they are completed. The app focuses on simplicity and helps users organize their daily activities effectively.

Features

Add new goals/tasks

Display tasks in a list

Delete tasks by tapping on them (onPress)

Clean and user-friendly interface

Technologies Used

React Native

Expo

JavaScript

Installation & Run

Clone the repository:

git clone https://github.com/your-username/your-repo-name.git


Go to the project folder:

cd your-repo-name


Install dependencies:

npm install


Start the Expo development server:

npx expo start


Run the app on:

Expo Go (mobile)

Android Emulator

iOS Simulator
